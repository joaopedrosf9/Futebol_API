Este projeto é um assistente virtual autônomo integrado ao Telegram, desenvolvido inteiramente em Python, que entrega dados estatísticos atualizados do Campeonato Brasileiro Série A diretamente na palma da mão do usuário. Através de comandos simples de chat (como /tabela e /artilheiros), o bot consome, processa e formata informações em tempo real provenientes de uma REST API esportiva (football-data.org), eliminando a necessidade de buscar estatísticas manualmente em sites ou aplicativos pesados.
